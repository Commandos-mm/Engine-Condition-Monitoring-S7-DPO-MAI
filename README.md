# Как запускать

Сменить директорию на app.
```
cd app
```

Собрать docker образ с решением можно при помощи команды: 

```
make build
```

Установить необходимые зависимости локально (**Необходимо наличие pip 3 версии**):
```
make install
```

Собрать приложение локально:
```
make run-local
```
